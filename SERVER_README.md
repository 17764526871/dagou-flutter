# Dagou AI 模型服务器

## 快速开始

### 方式一：双击启动（Windows）

直接双击 `启动服务器.bat` 文件即可启动服务器。

### 方式二：命令行启动

```bash
node model_server.js
```

## 服务器信息

启动后会显示：
- 📁 模型目录位置
- 🌐 服务器端口
- ✅ 可用模型列表
- 📱 局域网访问地址（手机使用）

## 手机端使用

1. 确保手机和电脑连接同一WiFi
2. 打开 Dagou AI 应用
3. 进入"模型管理"
4. 输入服务器地址（例如：`192.168.31.50:8080`）
5. 点击"连接服务器"
6. 选择模型并下载

## 下载模型到电脑

如果电脑上还没有模型文件：

```bash
node download_models.js
```

## 详细文档

查看 [docs/模型管理完整指南.md](docs/模型管理完整指南.md) 了解更多信息。

## 技术支持

- 默认端口：8080
- 模型目录：U:\DG model
- API接口：GET /api/models

## 注意事项

- 仅在局域网使用，不要暴露到公网
- 确保防火墙允许8080端口
- 模型文件较大，下载需要时间
