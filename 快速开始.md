# 快速开始指南

## 🚀 5分钟快速开始

### 步骤1：配置Git LFS（首次）

**Windows用户：**
```bash
# 双击运行
setup_lfs.bat
```

**Mac/Linux用户：**
```bash
# 运行脚本
bash setup_lfs.sh
```

### 步骤2：添加模型文件

```bash
# 添加配置文件
git add .gitattributes
git commit -m "chore: 添加Git LFS配置"

# 添加模型文件（2.5GB，需要时间）
git add assets/models/gemma-4-E2B-it.litertlm
git commit -m "feat: 添加Gemma 4 E2B模型"

# 推送到GitHub（首次较慢）
git push
```

### 步骤3：构建APK

**方式A：GitHub Actions（推荐）**
1. 访问 GitHub → Actions
2. 选择 "Manual Build"
3. 点击 "Run workflow"
4. 选择 "android"
5. 等待 5-8 分钟
6. 下载 Artifacts

**方式B：本地构建**
```bash
flutter build apk --release
```

### 步骤4：安装测试

```bash
# 通过ADB安装
adb install build/app/outputs/flutter-apk/app-release.apk

# 或传输到手机后安装
```

## ⚠️ 重要提示

### Android
- ✅ 完全支持
- ✅ APK包含模型，安装即用
- ✅ 需要4GB+内存设备

### iOS
- ❌ 暂不支持
- ❌ flutter_gemma插件限制
- ⏳ 等待官方更新
