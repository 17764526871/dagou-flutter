# Dagou AI 项目总结

## 项目概述

Dagou AI 是一个基于 Flutter 开发的跨平台多模态 AI 助手应用，集成了 Google 最新的 Gemini/Gemma 4 模型，支持文字、图片、视频和语音等多种交互方式。

## 技术架构

### 前端框架
- **Flutter 3.41.7** - 跨平台 UI 框架
- **Dart 3.11.5** - 编程语言
- **Material Design 3** - UI 设计规范

### AI 模型
- **Google Generative AI SDK** - AI 模型接口
- **Gemini 2.0 Flash** - 支持多模态的最新模型
- 支持文本、图片、视频等多种输入

### 核心依赖
```yaml
google_generative_ai: ^0.4.6  # AI 模型
camera: ^0.11.0+2              # 相机功能
image_picker: ^1.1.2           # 图片选择
video_player: ^2.9.1           # 视频播放
video_thumbnail: ^0.5.3        # 视频缩略图
speech_to_text: ^7.0.0         # 语音识别
flutter_tts: ^4.1.0            # 语音合成
record: ^5.1.2                 # 录音功能
```

## 功能特性

### 1. 文字对话
- 实时对话交互
- 上下文记忆
- Markdown 格式支持

### 2. 图片理解
- 相册选择图片
- 实时拍照
- 自动压缩优化（1024x1024）
- 图片内容分析

### 3. 视频分析
- 视频文件上传
- 自动提取关键帧
- 画面内容理解

### 4. 语音交互
- 语音转文字输入
- 中文语音识别
- AI 回复语音播报
- 可调节语速和音量

### 5. UI/UX
- Material Design 3 设计
- 深色/浅色主题自动切换
- 流畅的动画效果
- 响应式布局

## 支持平台

- ✅ **Android** 5.0+ (API 21+)
- ✅ **iOS** 12.0+
- ✅ **Windows** 10+
- ✅ **鸿蒙** HarmonyOS NEXT (理论支持)

## 性能优化

### 图片处理
- 自动压缩到 1024x1024
- JPEG 质量 85%
- 减少网络传输

### 视频处理
- 仅提取关键帧
- 避免完整视频上传
- 降低 API 调用成本

### 内存管理
- 及时释放资源
- 图片缓存优化
- 对话历史管理

## 安全性

### API Key 保护
- 配置文件不提交到 Git
- 使用环境变量或配置文件
- 支持本地模型推理（未来）

### 权限管理
- 运行时权限请求
- 最小权限原则
- 用户可控

### 数据隐私
- 不存储用户对话
- 不上传敏感信息
- 符合隐私规范
