## 🎯 使用指南

### 基本操作
1. **发送消息**: 输入文字后点击发送或按回车
2. **语音输入**: 点击麦克风切换，长按说话
3. **图片分析**: 选择图片后输入问题再发送
4. **取消发送**: 点击右上角取消按钮

### 高级功能
1. **系统提示词**: 设置 → 系统提示词 → 输入或选择预设
2. **翻译模式**: 设置 → 翻译功能 → 启用并选择方向
3. **性能监控**: 设置 → 显示设置 → 显示性能指标
4. **语音播报**: 点击消息下方喇叭图标

### 快捷操作
- 长按录音: 语音输入
- 上滑取消: 取消录音
- 点击图片: 预览大图
- 点击喇叭: 语音播报
