# 使用说明

## 配置 API Key

1. 访问 [Google AI Studio](https://makersuite.google.com/app/apikey) 获取免费的 API Key
2. 打开 `lib/config/app_config.dart` 文件
3. 将 `YOUR_API_KEY_HERE` 替换为你的真实 API Key：

```dart
static const String googleAiApiKey = '你的API_KEY';
```

4. 保存文件并重新运行应用

## 功能使用

### 文字对话
直接在输入框输入文字，点击发送按钮即可与 AI 对话。

### 图片分析
1. 点击"图片"按钮从相册选择图片
2. 或点击"拍照"按钮直接拍照
3. AI 会自动分析图片内容并回复

### 视频分析
1. 点击"视频"按钮选择视频
2. 应用会自动提取关键帧
3. AI 分析视频画面内容

### 语音输入
1. 点击"语音"按钮开始录音
2. 说话完毕后再次点击停止
3. 语音会自动转换为文字并发送

### 语音播报
AI 的回复会自动通过语音播报出来。

## 权限授予

首次使用时，应用会请求以下权限：
- **相机权限** - 用于拍照功能
- **麦克风权限** - 用于语音输入
- **存储权限** - 用于读取相册图片和视频

请点击"允许"以启用完整功能。

## 常见问题

### 提示"AI 服务未初始化"
- 检查是否已配置有效的 API Key
- 确保设备已连接网络
- 重启应用

### 语音识别不工作
- 检查是否已授予麦克风权限
- 确保设备支持语音识别
- 尝试在设置中启用语音输入

### 图片上传失败
- 检查是否已授予存储权限
- 确保图片格式正确（支持 JPG、PNG）
- 图片会自动压缩到 1024x1024

### 应用运行缓慢
- 清空对话历史（点击右上角删除按钮）
- 关闭其他后台应用
- 检查网络连接速度

## 模型选择

在 `lib/config/app_config.dart` 中可以切换不同的模型：

```dart
static const String defaultModel = 'gemini-2.0-flash-exp';
```

可选模型：
- `gemini-2.0-flash-exp` - 最新，支持多模态（推荐）
- `gemini-1.5-pro` - 高质量，长上下文
- `gemini-1.5-flash` - 快速响应
