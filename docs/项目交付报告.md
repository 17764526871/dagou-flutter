## 📝 配置指南

### 必需配置：API Key

**步骤 1**: 获取 API Key
- 访问: https://makersuite.google.com/app/apikey
- 登录 Google 账号
- 创建新的 API Key（免费）

**步骤 2**: 配置到项目
```bash
# 编辑配置文件
notepad lib\config\app_config.dart

# 找到这一行：
static const String googleAiApiKey = 'YOUR_API_KEY_HERE';

# 替换为你的真实 API Key：
static const String googleAiApiKey = 'AIzaSy...你的密钥';
```

**步骤 3**: 重启应用
```bash
# 在运行的应用中按 R 键热重启
# 或重新运行 flutter run
```
