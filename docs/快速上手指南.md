# 🚀 5 分钟快速上手指南

## 第一步：配置 API Key（2 分钟）

### 1. 获取 Google AI API Key
访问：https://makersuite.google.com/app/apikey
- 登录 Google 账号
- 点击 "Create API Key"
- 复制生成的密钥

### 2. 配置到项目
```bash
notepad lib\config\app_config.dart
```

找到第 11 行，替换：
```dart
static const String googleAiApiKey = 'YOUR_API_KEY_HERE';
```
改为：
```dart
static const String googleAiApiKey = '你的API密钥';
```

保存文件。

---

## 第二步：运行应用（1 分钟）

### Windows 版本（推荐，最快）
```bash
flutter run -d windows
```

### Android 版本（如果 Gradle 已完成）
```bash
flutter run -d 23113RKC6C
```

---

## 第三步：测试功能（2 分钟）

### 1. 文字对话
- 在输入框输入："你好，介绍一下你自己"
- 点击发送按钮
- 查看 AI 回复

### 2. 图片分析
- 点击"图片"按钮
- 选择一张图片
- AI 会自动分析图片内容

### 3. 语音输入
- 点击"语音"按钮（变红表示开始录音）
- 说话
- 再次点击停止
- 语音会转换为文字并发送

## 常见问题

### Q: 提示"AI 服务未初始化"？
**A**: 检查 API Key 是否正确配置，按 R 键重启应用。

### Q: Android 构建失败？
**A**: 先使用 Windows 版本测试，功能完全相同。

### Q: 没有 API Key？
**A**: 访问 https://makersuite.google.com/app/apikey 免费获取。
