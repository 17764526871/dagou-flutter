## 🚀 快速开始

### 第一步: 配置 API Key
```bash
notepad lib\config\app_config.dart
# 替换 YOUR_API_KEY_HERE
# 获取: https://makersuite.google.com/app/apikey
```

### 第二步: 运行应用
```bash
# Windows（推荐）
run-windows.bat

# Android（需解决 Gradle 问题）
deploy-android.bat
```

### 第三步: 查看文档
```bash
# 打开文档索引
notepad docs\README.md
```

## ⚠️ 已知问题

### Android 部署问题

**问题**: Gradle 下载因网络问题失败

**影响**: 无法直接在 Android 设备上运行

**解决方案**:
1. ✅ 使用 Windows 版本（推荐）
2. ✅ 手动下载 Gradle
3. ✅ 使用 Android Studio
4. ✅ 降级 Gradle 版本
5. ✅ 使用代理或 VPN

**详细指南**: 查看 [Android部署问题解决.md](Android部署问题解决.md)
